Development

Versions

There are two development branches.

v2.0.x, based on the current stable release, contains all the latest bugfixes. It should generally be stable, but is less well tested than a release version. If you want to stay up-to-date with all the latest fixes, choose this version.

v2.1 is the development version of the next release. In this version new features are implemented. It is functional but likely be very buggy. However, if you want to see where the program is headed, this is the version for you.

Download

  • v2.0.x - development version of the current stable release

    Download snapshot: .zip | .tar.gz

    Download using git: git clone http://github.com/lcts/spincounting.git

  • v2.1 - development version of the next release

    Download snapshot: .zip | .tar.gz

    Download using git: git clone http://github.com/lcts/spincounting.git --branch develop
