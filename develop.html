Development

Get development versions

There are two development versions available:

  • Dev version of the current stable version (v2.0.x). Contains all the latest bugfixes, but might not be entirely stable - Download | .zip | .tar.gz |

  • Dev version of the current development version (v2.1). Contains all the newest features, but is very untested - Download | .zip | .tar.gz |

If you want to stay up-to-date with development, I suggest you use git:

git clone http://github.com/lcts/spincounting (v2.0.x)

git clone http://github.com/lcts/spincounting --branch develop (v2.1)
