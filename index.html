Important note to users of v1.x

There have been some changes to the API that require you to update your scripts:

  • tfactor's definition changed (it now follows the definition in Eaton & Eaton - Quantitative EPR). Adapt your old calibration factors according to tfactor_v2 = 1 / tfactor_v2.
  • the parameter structs intparams and qparams have been replaced by individual options
    • qparams.background → tunebglimits
    • qparams.smoothing → tunepicsmoothing
    • qparams.order → tunebgorder
    • qparams.dipmodel → dipmodel
    • intparams.background → intbglimits
    • intparams.order → intbgorder
  • the default order for background correction during integration is now [1 3]

Overview

This MatLab toolbox is designed to facilitate and automate the evaluation of quantitative EPR data. In can be used both to calibrate an EPR spectrometer for future quantitative measurements and to calculate the number of spins from a spectrum obtained on a calibrated machine.

Additionally, when given a file containing a tune picture, it can perform automatic calculation of the quality factor Q.

Requirements

  • MatLab 2012 or later
  • MatLab Optimisation Toolbox
  • easyspin's eprload() function for import of Bruker Xepr files

Installation & first steps

Extract the downloaded archive to any location on your computer, and add that location to your MatLab search path. (In MatLab → 'Set Path ...' or by editing MatLab's pathdef.m). The toolbox can be invoked by typing spincounting on the MatLab command line. Further information can be found in the documentation/ and examples/ subfolders.
